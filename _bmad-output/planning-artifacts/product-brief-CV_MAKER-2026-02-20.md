# Product Brief: CVKraft

## Executive Summary

**CVKraft** adalah platform web gratis untuk membuat CV yang ATS-friendly dengan mudah dan cepat. Platform ini mengatasi masalah utama pencari kerja — dari fresh graduate hingga profesional berpengalaman — yang kesulitan membuat CV yang mampu lolos sistem ATS (Applicant Tracking System) perusahaan. Dengan pendekatan guided form, live preview, dan fitur ATS Score, CVKraft memastikan setiap CV yang dihasilkan memiliki format bersih, section lengkap, dan keyword yang relevan. CVKraft mendukung multi-bahasa (Indonesia & English) dan sepenuhnya gratis tanpa biaya tersembunyi untuk export PDF.

---

## Core Vision

### Problem Statement

Mayoritas pencari kerja membuat CV menggunakan tools seperti Canva yang menghasilkan desain menarik secara visual, namun tidak ATS-friendly — menggunakan grafik, layout multi-kolom, dan elemen visual yang tidak dapat dibaca oleh sistem ATS. Ditambah lagi, platform CV maker yang ada (Zety, Resume.io, Novoresume) umumnya menerapkan model freemium di mana fitur esensial seperti export PDF dikenakan biaya. Akibatnya, banyak CV bagus yang tereliminasi secara otomatis sebelum sampai ke tangan HR.

### Problem Impact

- **Fresh graduate** kehilangan kesempatan interview pertama karena CV tidak lolos filter ATS
- **Profesional berpengalaman** yang ingin pindah kerja tidak mendapat panggilan meskipun kualifikasi memadai
- **Freelancer** kesulitan mempresentasikan portfolio dan pengalaman dalam format yang diterima sistem rekrutmen
- Kurangnya edukasi tentang ATS membuat pencari kerja tidak menyadari CV mereka bermasalah dari segi format

### Why Existing Solutions Fall Short

| Solusi Existing | Kelemahan |
|----------------|-----------|
| **Canva** | Template menarik tapi tidak ATS-friendly (grafik, kolom kompleks, text-as-image). Template premium berbayar |
| **Zety, Resume.io, Novoresume** | Freemium — export PDF berbayar, fitur penting dikunci di balik paywall |
| **Microsoft Word manual** | Tidak ada panduan ATS, formatting inconsistent, memakan waktu |
| **Template internet gratis** | Kualitas tidak terjamin, sering tidak ATS-compliant, tidak ada feedback |

### Proposed Solution

**CVKraft** menyediakan platform web dengan:
- **Guided form step-by-step** — user tinggal mengisi form terstruktur tanpa perlu memikirkan layout
- **Live preview real-time** — setiap perubahan langsung terlihat di preview CV
- **Template ATS-friendly** — semua template dirancang khusus untuk lolos ATS (font standar, layout clean, section terstruktur)
- **Export PDF gratis** — text-based PDF yang bisa dibaca mesin ATS
- **ATS Score** — analisa CV berdasarkan format compliance (30%), section completeness (25%), keyword match (25%), dan content quality (20%)
- **Job Description Keyword Match** — user paste job description, CVKraft analisa dan sarankan keyword yang perlu ditambahkan
- **Tips pengisian konten** — panduan di setiap section untuk membantu user menulis konten yang efektif
- **Multi-bahasa** — support CV dalam Bahasa Indonesia dan English
- **Simpan & edit kembali** — user bisa login, simpan CV, dan kembali mengedit kapan saja

### Key Differentiators

1. **100% Gratis** — tidak ada biaya untuk export PDF atau fitur apapun, berbeda dari kompetitor yang freemium
2. **ATS-First Approach** — setiap template dan output dirancang khusus untuk ATS compliance, bukan sekadar "cantik di mata"
3. **ATS Score + Keyword Match** — fitur analisa yang memberikan feedback actionable, termasuk pencocokan dengan job description
4. **Edukasi ATS built-in** — user tidak hanya membuat CV tapi juga memahami kenapa format ATS penting
5. **Multi-bahasa (ID + EN)** — melayani pasar Indonesia yang belum banyak di-serve oleh kompetitor global
6. **General-purpose** — bisa digunakan siapapun dari bidang apapun

## Target Users

### Primary Users

#### Persona 1: Rina — Fresh Graduate
- **Profil:** Perempuan, 22 tahun, baru lulus S1 Manajemen. Belum punya pengalaman kerja formal selain magang 1 semester
- **Konteks:** Sedang aktif apply ke 10-20 lowongan per minggu lewat JobStreet, LinkedIn, dan Glints
- **Pain Points:**
  - Tidak tahu apa yang harus ditulis di CV karena minim pengalaman
  - CV yang dibuat di Canva terlihat bagus tapi tidak pernah dipanggil interview
  - Tidak paham konsep ATS dan kenapa CV-nya tidak pernah lolos
  - Bingung cara menyusun section yang tepat tanpa pengalaman kerja
- **Motivasi:** Mendapatkan panggilan interview pertama
- **Success Moment:** "Ternyata CV saya selama ini tidak lolos ATS! Setelah pakai CVKraft, minggu ini saya dapat 3 panggilan interview"

#### Persona 2: Andi — Career Switcher (Profesional Berpengalaman)
- **Profil:** Laki-laki, 30 tahun, 5 tahun pengalaman sebagai Marketing Executive, ingin pindah ke perusahaan yang lebih besar
- **Konteks:** Punya CV lama format Word yang sudah outdated, ingin update dengan achievement terbaru atau bikin ulang dari nol
- **Pain Points:**
  - CV lama formatnya berantakan dan tidak menonjolkan achievement
  - Tidak tahu cara menyesuaikan keyword CV dengan job description target
  - Pernah pakai Resume.io tapi harus bayar untuk download PDF
  - Butuh CV yang cepat jadi karena ada deadline apply
- **Motivasi:** Mendapatkan posisi yang lebih baik dengan gaji lebih tinggi
- **Success Moment:** "CVKraft bantu saya cocokkan keyword dengan job description. ATS Score naik dari 45 ke 92, dan saya dapat interview di perusahaan impian"

### Secondary Users

#### Persona 3: Dimas — Freelancer
- **Profil:** Laki-laki, 27 tahun, freelance graphic designer 3 tahun, sesekali butuh CV formal untuk apply ke perusahaan atau project korporat
- **Konteks:** Biasanya pakai portfolio, tapi beberapa klien korporat dan tender minta CV formal
- **Pain Points:**
  - Tidak terbiasa bikin CV formal, lebih sering kirim portfolio
  - Bingung cara menyajikan pengalaman freelance agar terlihat profesional
- **Motivasi:** Punya CV formal yang siap kapanpun dibutuhkan
- **Success Moment:** "Akhirnya punya CV profesional yang bisa saya kirim kapan saja tanpa ribet format ulang"

#### Persona 4: Maya — Siswa SMK/SMA (Magang/Kerja Pertama)
- **Profil:** Perempuan, 17 tahun, siswa SMK jurusan Akuntansi, butuh CV untuk melamar magang atau kerja part-time
- **Konteks:** Pertama kali membuat CV, benar-benar tidak tahu harus mulai dari mana
- **Pain Points:**
  - Belum punya pengalaman apapun selain organisasi sekolah
  - Tidak punya akses ke tools berbayar
  - Guru hanya memberikan contoh CV yang sudah ketinggalan zaman
- **Motivasi:** Punya CV yang layak untuk melamar magang
- **Success Moment:** "CVKraft kasih tips apa yang bisa ditulis meski belum punya pengalaman kerja. Sekarang CV saya terlihat profesional!"

## MVP Scope

### Core Features (Must Have)

| # | Fitur | Deskripsi | Prioritas |
|---|-------|-----------|-----------|
| 1 | **User Authentication** | Register/login dengan email + Google OAuth. Simpan CV per akun | P0 |
| 2 | **Guided Form Input** | Form step-by-step: Data Pribadi → Ringkasan → Pengalaman → Pendidikan → Keahlian → Sertifikasi → Proyek | P0 |
| 3 | **Live Preview** | Preview CV real-time di samping form, update setiap perubahan field | P0 |
| 4 | **ATS-Friendly Templates** | 3-5 template clean single-column (Professional, Modern, Minimal, Classic, Fresh Graduate) | P0 |
| 5 | **PDF Export (Gratis)** | Generate text-based PDF yang ATS-parsable, tanpa biaya | P0 |
| 6 | **ATS Score** | Analisa otomatis: format compliance (30%), section completeness (25%), keyword match (25%), content quality (20%) | P0 |
| 7 | **Job Description Keyword Match** | User paste job description, sistem analisa & sarankan keyword yang perlu ditambahkan | P1 |
| 8 | **Content Tips** | Tooltip/panduan di setiap field form untuk membantu user menulis konten efektif | P1 |
| 9 | **Multi-bahasa CV** | Output CV support Bahasa Indonesia dan English (user pilih bahasa output) | P1 |
| 10 | **Dashboard CV** | Halaman untuk manage semua CV yang sudah dibuat, edit, duplicate, atau hapus | P1 |
| 11 | **Landing Page** | Homepage yang menjelaskan CVKraft, value proposition, dan CTA untuk mulai bikin CV | P1 |

### Out of Scope (MVP)

| Fitur | Alasan Ditunda | Target |
|-------|---------------|--------|
| **AI Content Generator** | Kompleksitas tinggi, butuh integrasi LLM, biaya API | v2.0 |
| **Import CV dari LinkedIn/PDF** | Parsing kompleks, banyak edge case | v2.0 |
| **Cover Letter Builder** | Fokus CV dulu, cover letter bisa jadi add-on | v2.0 |
| **Collaboration/Sharing** | Fitur sosial belum prioritas untuk MVP | v2.0 |
| **Mobile App (Native)** | Web responsive cukup untuk MVP | v3.0 |
| **Subscription/Payment System** | Gratis dulu, monetisasi setelah product-market fit | Post-MVP |
| **Admin Dashboard/Analytics** | Cukup pakai analytics tool external (GA/Posthog) dulu | v2.0 |
| **Multiple CV Versions per Job** | Nice-to-have, MVP cukup duplicate manual | v1.5 |

### MVP Success Criteria

| Gate | Kriteria | Threshold |
|------|----------|-----------|
| **User Adoption** | Registered users dalam 3 bulan pertama | > 500 users |
| **Problem Validation** | CV completion rate (mulai → download PDF) | > 50% |
| **Value Delivery** | Rata-rata ATS Score CV yang di-generate | > 75/100 |
| **Retention Signal** | User yang kembali dalam 30 hari | > 20% |
| **Technical Stability** | Uptime dan error rate | 99% uptime, < 1% error |
| **Go/No-Go Decision** | Jika 3 dari 5 gate terpenuhi → proceed to v1.5 | 3 bulan post-launch |
